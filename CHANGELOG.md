# Changelog

이 문서는 사용자에게 노출되는 릴리즈 변경 사항을 기록합니다.

## [1.0.1] - 2026-04-27

### Added

- 예약 수정 시 회의실/회의테이블을 변경할 수 있도록 지원
- 관리자 패널에서 예약 라벨을 삭제하지 않고 숨김/보임 상태로 관리하는 기능 추가

### Changed

- 회의실/회의테이블 변경 시 기존 시간 선택을 초기화해 새 장소의 예약 겹침을 다시 확인하도록 개선
- 숨긴 라벨은 새 예약 선택지에서 제외하되, 기존 예약에 붙은 라벨은 그대로 표시되도록 변경
- 새 예약 다이얼로그를 열면 라벨 기본값이 항상 `없음`으로 선택되도록 조정

### Fixed

- 사용자 이름 메뉴가 열린 상태에서 다른 화면을 누르면 닫히도록 수정
- 회의록 불릿 입력과 줄바꿈 동작을 정리하고, Shift+Enter 줄바꿈 위치를 본문 시작점에 맞춤
- 예약카드 주요 안건 입력도 `- `로만 불릿으로 변환되고, `Enter`/`Shift+Enter` 동작을 회의록 페이지와 같게 정리
- 예약카드의 라벨/내부 참석자 입력 박스가 화면 축소 시 잘리지 않도록 보완
- 내부 참석자 입력 시 이름을 입력하고 Enter를 누르면 자동완성 첫 번째 후보가 바로 선택되도록 개선
- 내부/외부 참석자 토큰 입력칸이 비어 있을 때 Backspace로 마지막 참석자를 바로 지울 수 있도록 보완
- 한글 입력 조합 중 Enter는 자동선택으로 처리되지 않도록 보완

## [1.0.0] - 2026-04-24

### Added

- 주간/월간 타임테이블 기반 회의실 예약 기능
- 회의록 작성과 회의록 Wiki 조회 기능
- 관리자용 사용자 및 예약 라벨 관리 기능

### Changed

- 사내 운영 기준에 맞춘 로그인 및 세션 유지 흐름 정리
- 회의 상세 정보와 참석자 입력 구조 개선

### Fixed

- 예약 충돌 시 재조회 기반으로 UI가 다시 동기화되도록 보완
- 최근 회의록 편집과 미리보기 흐름에서 발생한 예외 처리 보강

### Known Issues

- 업데이트 내역은 현재 앱 내부에서 최신 릴리즈부터 확인할 수 있습니다.
- 배포 직후 브라우저 캐시가 남아 있으면 새 버전 표기가 한 번 늦게 보일 수 있습니다.
